Sitio Web LalyVet

Desarrollé un proyecto de programación web llamado LalyVet, diseñado para una clínica veterinaria. La plataforma incluye secciones como Inicio, Servicios, Quiénes somos y Contáctanos, facilitando la interacción con los clientes y la gestión de sus mascotas. En el desarrollo, utilicé HTML, CSS y JavaScript para crear una interfaz atractiva y funcional, permitiendo a los usuarios acceder a información sobre los servicios ofrecidos y realizar consultas fácilmente. Este proyecto me brindó la oportunidad de aplicar mis habilidades en programación web y entender la importancia de ofrecer una experiencia de usuario intuitiva.

Portafolio Personal

Desarrollé un proyecto de portafolio personal en formato de página web para presentar mi información profesional y académica de manera organizada y visualmente atractiva. La página incluye secciones clave como Información personal, Estudios, Portafolio y Contacto, proporcionando a los visitantes una visión completa de mis habilidades, proyectos previos y trayectoria académica. En este proyecto, utilicé HTML, CSS y JavaScript para diseñar una interfaz intuitiva y adaptable que permite una navegación fluida. Este portafolio no solo me permite mostrar mi trabajo y experiencias, sino que también refleja mi estilo y dedicación en el desarrollo web.
